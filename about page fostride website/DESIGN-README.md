# Fostride Website - Pencil Sketch Design

## Overview
This is a complete recreation of the Fostride website (www.fostride.com) with a hand-drawn pencil sketch aesthetic. The design maintains all the original content and structure while transforming the visual style to look like it was drawn by hand.

## Key Design Features

### 1. **Color Palette**
- **Background**: Paper-like beige (#f5f3ed) with subtle grid texture
- **Text**: Pencil dark (#2a2a2a) and lighter gray (#6b6b6b)
- **Accent**: Muted green (#4a7c59) for eco-friendly theme
- **Cards**: White (#ffffff) with dark borders

### 2. **Typography**
- **Headings**: Caveat (handwritten, cursive font)
- **Body Text**: Kalam (hand-drawn sans-serif)
- **Weights**: Bold (700-900) for emphasis

### 3. **Hand-Drawn Effects**
- **Borders**: 2.5-3px solid borders instead of thin lines
- **Shadows**: Offset box-shadows (6-10px) for depth
- **Rotations**: Slight random rotations (-2deg to 2deg) on elements
- **Sketchy Highlights**: Diagonal line patterns for text emphasis

### 4. **Interactive Elements**
- **Buttons**: Thick borders, box shadows, slight rotations
- **Hover Effects**: Straighten rotation, increase shadow
- **Cards**: Rotate on hover, lift with shadow
- **Links**: Underline animations

### 5. **Sections Styled**

#### Navigation
- Fixed header with paper background
- Thick bottom border (3px)
- Handwritten logo with rotation
- Underline hover effects on links

#### Hero Section
- Large handwritten heading (4.5rem)
- Rotated image container with thick border
- Call-to-action button with sketchy style

#### Service Cards
- White cards with 3px borders
- Alternating rotations (0.5deg, -0.8deg)
- Grayscale emoji icons
- Box shadow depth

#### Testimonial
- Large outlined background text
- White card with rotation
- Sketchy avatar with border
- Slider dots with borders

#### Bento Grid (What Makes Us Special)
- Mixed layout with rotations
- Image with sepia filter
- Green accent card
- Thick borders throughout

#### Blog Section
- Rotated cards with borders
- Images with bottom border separator
- Handwritten tags and headings

#### FAQ
- Large handwritten heading
- Thick border separators
- Rotating + icon on expand
- Hover color change

#### Footer
- Light beige background (#e8e6dc)
- Thick top border
- Circular social icons with borders
- Handwritten headings

### 6. **Responsive Design**
- Mobile-friendly breakpoints
- Adjusted font sizes for smaller screens
- Single column layouts on mobile
- Maintained sketch aesthetic across devices

## Files Structure

## Browser Compatibility
- Modern browsers (Chrome, Firefox, Safari, Edge)
- CSS Grid and Flexbox support required
- Google Fonts (Caveat, Kalam) loaded from CDN

## Design Philosophy
The pencil sketch style creates a warm, approachable, and eco-friendly aesthetic that aligns perfectly with Fostride's sustainability mission. The hand-drawn look suggests authenticity, creativity, and a human touch in waste management solutions.
